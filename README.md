# dotfiles Configuration

## Installation

Clone this repository to `~/.config/`.

### neovim

Install [lazy.nvim](https://lazy.folke.io/installation).

### kitty

Ensure the [JetBrainsMono Nerd
Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/JetBrainsMono.zip)
is present in your font configuration.

### zsh

Download [zsh-async](https://github.com/mafredri/zsh-async):

```sh
mkdir -p ~/.local/share/zsh/site-functions/
curl -L https://raw.githubusercontent.com/mafredri/zsh-async/refs/heads/main/async.zsh -o ~/.local/share/zsh/site-functions/async
```
